| build system → foreign-target artifacts | Produced binaries target other OS/arch and are not trusted to run here |

| T-08-07 | Elevation of Privilege | executing an untrusted foreign-target binary on the host | medium | mitigate | Build-only: assert the output file exists and is non-empty; never invoke non-host binaries (D-05) |
| T-08-08 | Denial of Service | test hangs or hard-fails when zig is absent | low | mitigate | zig_available() guard returns early with a skip message (matches existing aot_cross test); suite passes without zig |
